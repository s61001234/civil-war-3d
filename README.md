# 國共內戰互動歷史地圖 (1927–1949)

Interactive historical map of the Chinese Civil War, 1927–1949.

🔗 **Live: https://s61001234.github.io/civil-war-3d/**

## 功能 Features

- 時間軸滑桿：九幕呈現 1927–1949 全程（北伐、清黨、五次圍剿、長征、西安事變、抗戰、全面內戰、三大戰役、渡江戰役）
- 各時期控制區色塊（國民政府＝藍、中共＝紅、日佔＝灰、軍閥＝褐）
- 戰役 ⭐、軍團兵力、將領簡介、移動箭頭，點擊可看詳情
- 「史實／俗稱」名稱切換（如 淮海戰役 ↔ 徐蚌會戰）
- 中文（繁體）／英文介面切換、自動播放模式
- 單一 HTML 檔案，無外部依賴，支援手機與桌面瀏覽器

## 注意 Caveats

- 省界採現代行政區劃近似，與民國時期省區不同
- 控制區為示意性概括；兵力數字取通行估計值，國共雙方記載分歧處已在說明框標注

Built as a single self-contained `index.html` (embedded simplified province GeoJSON, no API keys, no external tiles).
